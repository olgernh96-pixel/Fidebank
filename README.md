Proyecto hecho en java con netbeans.
Funcionalidades:
Crear ceuntas y clientes.
hacer depositos, retiros, y transferencias.
Este programa su objetivo principal es la automatizacion de un cajero de un banco mediante un programa 
Para ejecutarlo solo debes ejecutarlo e iniciara con la pantalla de inicio donde, tendras la opcion de login 
Si no tienes usuario en la esquina inferior derecha existe un boton que te redirije a otra ventana para crear usuario.
Una vez creado el usuario podes hacer login y entraras al menu principal donde tenes varias opciones de transacciones disponibles.
Depositar dinero efectivo en diferentes cuentas.
Retirar dinero en efectivo de tu cuenta.
Realizar transferencias entre cuentas del mismo banco.
Ver movimientos de transacciones de tu cuenta.
Boton de saldo de tu cuenta(el cual tienes que presionar para actualizar el saldo)
